# Changelog

## v2.3.0 (2025-10-31)
---
* **重大改进：统一配置文件格式**
  - 统一使用 JSON 格式配置文件（`~/.minidict.json`）
  - 更新 README 文档，确保配置说明与代码一致
  - 完善默认配置项（showPhonetic, showExamples, maxExamples）

* **类型系统改进**
  - 修复 TranslationResult 类型定义，统一音标和例句格式
  - 音标支持字符串和对象格式（{uk?, us?}）
  - 例句统一为 {en, zh} 结构
  - 统一字段名：使用 pluginName 替代 source

* **配置选项完善**
  - `--max-examples` 选项现在真正生效
  - 命令行选项可以覆盖配置文件设置
  - 完善配置项的默认值处理

* **解析器健壮性增强**
  - 为必应词典解析器添加多个备选 CSS 选择器
  - 增强错误处理和边界情况处理
  - 添加翻译结果和例句的去重和过滤逻辑
  - 提高对网站结构变化的适应性

* **测试改进**
  - 更新所有测试用例以匹配新的代码结构
  - 修复测试导入路径
  - 使用 node-fetch mock 替代 axios mock
  - 删除不存在的插件测试（google）

* **构建和工程化改进**
  - 优化构建脚本，移除不必要的 touch 命令
  - 提升 Node.js 版本要求至 16.0.0（配合 node-fetch@3）
  - 改进错误处理和信息提示
