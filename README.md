# MiniDict

简单的mini字典，基于 Node 开发，支持 Linux、Mac、Windows。

主要功能：

use
- 中英互译
- 输出翻译、音标、例句，并且可自由配置

## 环境

运行 MiniDict 需要：

- Node ( **v10+** )
- NPM

安装方法请查看 Node 官网：[https://nodejs.org/](https://nodejs.org/)

## 安装

使用 npm 安装，执行：

```shell
npm i -g minidict
```

当然也可以使用 [yarn](https://yarnpkg.com)：

```shell
yarn global add minidict
```
